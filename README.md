# 🌱 PlantiFresh

Aplicativo mobile desenvolvido para facilitar o cuidado com plantas, oferecendo uma interface intuitiva e recursos para acompanhamento e gerenciamento.



##  Tecnologias Utilizadas

- React Native
- Expo SDK 54
- Expo Router
- TypeScript
- React Navigation
- Expo Vector Icons
- Feather

## 🚀 Como executar

### 1. Clone o repositório


git clone <https://github.com/VSLima-dj/Projeto-do-Glaube>

### 2. Entre na pasta


cd PlantiFresh


### 3. Instale as dependências


npm install


### 4. Execute o projeto


npx expo start

Depois disso:

- pressione **a** para Android
- pressione **w** para Web
- ou leia o QR Code com o **Expo Go**.
